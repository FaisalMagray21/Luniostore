A full-featured **multivendor e-commerce platform** built with the **MERN Stack (MongoDB, Express, React, Node.js)**. The platform supports multiple vendors, product management, secure authentication, and a responsive user interface.

---

## **Features**

- **User Authentication with Email Verification**:  
  Users receive a verification code via **NodeMailer** upon signup for secure account creation.
- **Product Management**: Vendors can upload products with **images using Multer**.
- **MVC Architecture**: Project follows the **Model-View-Controller (MVC)** architecture for clean and maintainable code.
- **Responsive Design**: Compatible with desktop and mobile devices.
- **RESTful APIs**: Fully functional backend supporting CRUD operations.
- **Scalable & Clean Code**: Focused on maintainability and best practices.

---

## **Technologies Used**

- **Frontend**: React.js, Tailwind CSS  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Others**: NodeMailer, Multer, JWT Authentication, MVC Architecture
